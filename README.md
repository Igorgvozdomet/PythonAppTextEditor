# Текстовый редактор

Простой и удобный текстовый редактор с графическим интерфейсом, написанный на Python с использованием фреймворка **Flet**. Позволяет открывать текстовые файлы, редактировать их и сохранять изменения.

---

## Функционал

| Кнопка | Что делает |
|--------|------------|
| **Выбрать файл** | Открывает диалог выбора файла, загружает его содержимое в редактор |
| **Сохранить** | Сохраняет текущий текст в открытый файл |

**Особенности:**
- Открытие существующих текстовых файлов
- Редактирование содержимого в многострочном поле
- Сохранение изменений обратно в файл
- Тёмная тема оформления
- Компактное окно (250×350)

---

## Технологии

- **Python** — язык программирования
- **Flet** — фреймворк для кроссплатформенных приложений
- **FilePicker** — нативный диалог выбора файлов
